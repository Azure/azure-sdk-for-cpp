std::string Azure::Identity::WorkloadIdentityCredentialOptions::AuthorityHost = _detail::DefaultOptionValues::GetAuthorityHost()
Note
Defaults to the value of the environment variable 'AZURE_AUTHORITY_HOST'. If that's not set, the default value is Microsoft Entra global authority (https://login.microsoftonline.com/).
Example of an authority host string: "https://login.microsoftonline.us/". See national clouds' Microsoft Entra authentication endpoints: https://learn.microsoft.com/azure/active-directory/develop/authentication-national-cloud.
