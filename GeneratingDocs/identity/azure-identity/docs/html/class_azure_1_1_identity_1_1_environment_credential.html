Detailed Description

Note
May read from the following environment variables:
  • AZURE_TENANT_ID
  • AZURE_CLIENT_ID
  • AZURE_CLIENT_SECRET
  • AZURE_CLIENT_CERTIFICATE_PATH
  • AZURE_CLIENT_CERTIFICATE_PASSWORD
  • AZURE_CLIENT_SEND_CERTIFICATE_CHAIN
  • AZURE_USERNAME
  • AZURE_PASSWORD
  • AZURE_AUTHORITY_HOST
