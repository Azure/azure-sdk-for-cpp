◆ EnableSystemDefaultProxy

bool Azure::Core::Http::WinHttpTransportOptions::EnableSystemDefaultProxy {false}

Proxy information.

Remarks
Set this to "true" if you would like to use a local HTTP proxy like "Fiddler" to capture and analyze HTTP traffic.

Set to "false" by default because it is not recommended to use a proxy for production and Fiddler's proxy interferes with the HTTP functional tests.

The Proxy Information string is composed of a set of elements formatted as follows:
