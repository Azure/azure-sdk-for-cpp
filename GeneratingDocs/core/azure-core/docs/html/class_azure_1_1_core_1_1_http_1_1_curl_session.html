Detailed Description

Remarks
This component does not use the classic libcurl easy interface to send and receive bytes from the network using callbacks. Instead, CurlSession supports working with the custom HTTP protocol option from libcurl to manually upload and download bytes from the network socket using curl_easy_send() and curl_easy_recv().
This component is expected to be used by an HTTP Transporter to ensure that transporter to be reusable in multiple pipelines while every call to network is unique.
